Lab 3:  ASCII-risks (Asterisks)
-----------
DESCRIPTION
In this lab, the user is prompted to input an int as input. If this input is below 1, then 
the prompt is repeated until this condition is satisfied.
When this condition is satisfied, a traingular pattern is printed to the screen with a 
height of the user input.
-----------
FILES
Lab3.asm
This file includes the assembly code to be run in MARS.
-----------
INSTRUCTIONS
This program is intended to be run using the MARS program. 
Open MARS and click open under file. Find Lab3.asm and select to use.
To run, assemble the code and click run.
