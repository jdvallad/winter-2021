Lab 1: Intro to Logic Simulation
-----------
DESCRIPTION
In this lab, the user can use 4 input switches to make a seven segment display and 
LED lights turn on or off depending on the user input.
-----------
FILES
Lab1.lgi
This file includes the MML pages with the circuits.
-----------
INSTRUCTIONS
This program is intended to be run using the MML program. 
Open the program and click Open under File and select Lab1.lgi to load the program.
